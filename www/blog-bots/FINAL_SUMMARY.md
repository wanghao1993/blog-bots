### 运行 UI 自动化测试

```bash
# 1. 安装 Playwright 浏览器（如果还没安装）
./setup-tests.sh

# 2. 运行测试（UI 模式，推荐）
pnpm test:ui

# 或者运行所有测试（无头模式）
pnpm test

# 查看测试报告
pnpm test:report
```
